## Cel projektu

Granisys porządkuje pięć kluczowych problemów teorii sygnałów:
- Regularność
- Skala
- Szum
- Miara‑0
- Predykcja

Celem jest stworzenie ramy systemowej, która działa zarówno jako opis rytmu, jak i narzędzie diagnostyczne do testowania systemów.
